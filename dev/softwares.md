## System

- [duti](https://github.com/moretension/duti) 修改软件默认打开方式

显示文件格式的默认打开方式
```shell
    duti -x .txt
```

修改文件的默认打开方式
```shell
    duti -s com.sublimetext.3 .php all
    duti -s com.sublimetext.3 .js all    
    ...
```
